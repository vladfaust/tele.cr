# Tele

A convenient Telegram Bot framework written in Crystal.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  tele:
    github: vladfaust/tele
    version: ~> 0.1.0
```

## Usage

```crystal
require "tele"
```

See `example/` for an example implementation.
